## ColorSwitch
 # This is a JavaFx Desktop Application which is similar to the popular Color Switch Game.
How to Play?
 1) Clone this repo.
 2) Open Project and Configure your JavaFX
 3) goto ColorSwitch-->src-->mainWindow-->MainPageMain, and run it.
